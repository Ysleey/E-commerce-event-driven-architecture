# KAN-36 - Documentacion tecnica consolidada

## Objetivo

Consolidar la documentacion tecnica del proyecto para facilitar onboarding, ejecucion local, arquitectura, contratos y roadmap evolutivo.

## Alcance implementado

1. Se crea una guia tecnica navegable del proyecto.
2. Se centraliza la documentacion de ejecucion local con Docker.
3. Se documentan APIs, eventos y seguridad.
4. Se describe la roadmap por epicas y dependencias.

## Evidencia tecnica

- Captura de Jira con la subtarea KAN-36 en estado de trabajo.
- DoD visible en Jira: documentacion revisada, navegable y alineada con el estado real.

## Definition of Done

- La documentacion tecnica queda publicada y util para cualquier desarrollador.
- Los enlaces internos y secciones principales quedan validados.
- El proyecto se puede levantar siguiendo la guia sin friccion.
