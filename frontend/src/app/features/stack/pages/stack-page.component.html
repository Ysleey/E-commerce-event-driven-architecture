Perfil profesional

Backend Software Developer y Líder Técnico apasionado por la ingeniería de software de calidad, el diseño eficiente y los sistemas escalables.

Mi experiencia principal se centra en el desarrollo de servicios críticos utilizando Java (17-21) y Spring Boot, optimizando la persistencia con MongoDB Atlas y diseñando APIs REST robustas bajo arquitecturas distribuidas (Hexagonal y Microservicios). Soy un firme defensor de la excelencia técnica, por lo que integro de forma natural el análisis de código limpio (Clean Code), principios SOLID y pruebas automatizadas con JUnit y Mockito, promoviendo prácticas de TDD para garantizar la resiliencia en producción.

Cuenta con certificacion en Kafka y esta finalizando certificacion en AWS, reforzando capacidades para arquitecturas cloud-native y event-driven.

Tengo un perfil versátil que me permite equilibrar el desarrollo de software de alto rendimiento con el liderazgo técnico de equipos. Me adapto con rapidez a ecosistemas complejos y tengo facilidad para traducir requerimientos de negocio abstractos (épicas e historias de usuario) en tareas técnicas atómicas, claras y priorizadas. Mi objetivo en cada proyecto es mantener al equipo cohesionado, asegurar una documentación impecable (Swagger/OpenAPI) y cumplir de forma rigurosa con los plazos de entrega bajo metodologías ágiles. Afronto los retos técnicos con una mentalidad analítica, resolutiva y con una gran iniciativa para investigar y adoptar nuevas tecnologías cuando el producto lo requiere.

Tech Stack & Competencias: • Backend: Java (17-21), Spring Boot, Spring Data, Microservicios, Maven. • Datos & Entorno: MongoDB Atlas, Git, Swagger/OpenAPI. • Calidad & Testing: JUnit 5, Mockito, TDD (Test-Driven Development), Code Review. • Patrones & Arquitectura: Arquitectura Hexagonal, Principios SOLID, Clean Code. • Liderazgo & Agilidad: Tech Lead, Gestión de carga de trabajo, Kanban, Jira.
